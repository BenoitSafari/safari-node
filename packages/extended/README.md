## Extended Package

Extended package is a collection of low-level utilities and helpers that extends default JS functionalities.

### Installation

```bash
npm install @safari-node/extended
```

### Usage

```typescript
import '@safari-node/extended';

JSON.safeParse('{"key": "value"}');
```
